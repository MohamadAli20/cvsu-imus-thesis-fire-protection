- HTML
- CSS
- EJS
- Express
- Node.js
- MVC Pattern

## Libraries
- Axios (HTTP request and response)
- Leaflet.js with OpenStreetMap
- Turf.js (Point-in Polygon - check if coordinates (lat, long) are within the polygon / area)
- D3.js (Visualization)
